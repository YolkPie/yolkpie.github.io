# vue 单页应用模板

Node 版本要求

Vue CLI 需要 Node.js 8.9 或更高版本 (推荐 8.11.0+)。你可以使用 nvm 或 nvm-windows 在同一台电脑中管理多个 Node 版本。

vue-spa-template 是基于 @vue/cli 搭建的 vue.js 单页应用快速开发的完整系统，提供：

  • @vue/cli：基于 @vue/cli 搭建该模板。
  • Vue：基于 vue.js。
  • VueX：集成 VueX，用于状态管理。
  • Vue Router：集成 Vue Router，用于路由管理。
  • axios：集成 axios，用于网络请求，支持：
    • 基于 @yolkpie/utils/axios-jsonp，支持发起 jsonp 请求。
    • 基于 axios-retry，自动重新发送请求。
  • mode：使用环境标量与模式，区分线上、预发、测试与开发环境。
  • Workbox：基于 Workbox，进行离线存储。
  • better-scroll：集成 better-scroll，处理移动端滚动。
  • postcss-pxtorem：基于 postcss-pxtorem，自动将px转成rem，计算规则为： 750px设计稿 -----> 7.5rem。
  • Prettier：基于 Prettier，进行代码格式化，保持团队的项目风格统一。
  • standard.js：基于 standard.js + husky，进行代码格式规格，统一团队代码风格，不符合规范㞏提交代码。
  • Commitizen：基于 Commitizen + husky，规范 commit message，不符合规范无法提交代码。
  • 骨架屏：抽取骨架屏效果到html，降低首屏空白时间。
  • 1px：伪元素 + transform 解决Retina屏1像素问题。
  • @yolkpie/utils：前端工具类库。
