Yolk CLI

前端项目脚手架

快速上手 →

使用简洁

仅需运行一行指令即可搭建一个完整的前端项目。

功能完备

对 Babel、ESLint、PostCSS、骨架屏等提供开箱即用的支持。

多技术栈

支持 react、vue 等技术栈，同时支持各种 css 预处理语言。

场景丰富

支持 PC 端及 M 端 H5 页面开发，支持单页应用与多页应用的开发。

灵活配置

各种项目模板完全是可配置的，如果现有脚手架的配置不能满足您的需求，您可进行扩展。

面向未来

为现代浏览器轻松产出原生的 ES2015 代码。

起步

安装：

npm install -g @yolkpie/yolk-cli
# OR
yarn global add @yolkpie/yolk-cli

创建一个项目：

yolk init
